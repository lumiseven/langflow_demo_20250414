# LangFlow Demo

## 启动 ppt(使用 slidev md -> slides)

- 使用 `slidev` 创建 slides

```sh
npm init slidev@latest
```

- 将 `slides.md` 放入 `slidev` 或软链接
- 启动 `cd slidev && npm run dev`
